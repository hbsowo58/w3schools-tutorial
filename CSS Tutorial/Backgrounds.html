The CSS background properties are used to define the background effects for elements. CSS background properties: background-color background-image background-repeat background-attachment background-position The background-color property specifies the background color of an element. With CSS, a color is most often specified by: a valid color name - like "red" a HEX value - like "#ff0000" an RGB value - like "rgb(255,0,0)" Look at CSS Color Values for a complete list of possible color values.
